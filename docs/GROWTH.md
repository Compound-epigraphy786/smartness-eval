# Growth Playbook (Stars & Awareness)

## Positioning sentence

"OpenClaw Smartness Eval is a reproducible, evidence-based scoring engine for AI agent capability evolution."

## Weekly growth loop

1. Publish one benchmark screenshot (`overall + CI + trend`).
2. Post one short write-up: what improved and why.
3. Share one practical tip from real telemetry analysis.
4. Link back to this repo and changelog.

## Star conversion tips

- Keep README hero section concise and metric-driven.
- Pin one before/after result in repo description.
- Add release notes with concrete value, not generic text.
- Include copy-paste quick start commands.

## Awareness channels

- GitHub Discussions / README updates
- X (Twitter) short benchmark cards
- Dev communities for AI agents / evaluation
- Demo clip in project homepage or pinned issue
