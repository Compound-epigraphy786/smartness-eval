## Typical use cases

- Weekly capability report for your OpenClaw instance
- Post-upgrade regression check (`--compare-last`)
- Confidence-backed result for team demos (`--llm-judge` optional)

## Example KPI snapshot

- `overall_score`: 71.36
- `grade`: B-

## Sharing format (recommended)

When sharing results publicly:
1. include score + confidence interval,
2. include trend delta vs last run,
3. include at least 3 evidence metrics,
4. mention environment and mode.
